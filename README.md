AI-Powered Fuel Quality and Engine Health Prediction System.

## 🚀 Features

- **Fuel Quality Analysis**: Real-time assessment of ethanol content, water contamination, and overall fuel quality
- **Engine Health Prediction**: ML-based prediction of component wear and maintenance needs
- **Interactive Dashboard**: Visual representation of fuel and engine metrics
- **Maintenance Recommendations**: Actionable insights for injector cleaning, fuel filter replacement, and more
- **Manual Data Injection**: Enter sensor and OBD-II data manually for testing and validation
- **Crowdsourced Intelligence**: Future capability for station ranking and fuel quality trends
